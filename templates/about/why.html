My Learning Journey and This Website

The Spark: Early 2024

In early 2024, while job hunting, I decided it was time for a change. Conversations with my wife and friends led me to pursue new learning opportunities. As a presales engineer specializing in Microsoft 365 with experience in Azure and AWS, I was drawn to the hot fields of Security and AI.

Choosing the Path: AI/ML Program

After exploring various options, I chose the Great Learning's PG-AI/ML Class. This seven-month program offered a comprehensive overview of AI/ML topics, backed by UT Austin's reputation. The journey began with learning Python, which I found surprisingly accessible and powerful compared to my college experience with C++ and Java.

From AI/ML to Web Development

Inspired by Python, I decided to explore web application development. This decision was the catalyst for this entire project. I wanted to create a platform to showcase my future applications, leading me to dive into web technologies:

  • Learned HTML, CSS, and JavaScript basics through The Odin Project
  • Explored hosting options, starting with a Raspberry Pi 5 at home
  • Transitioned to AWS free tier for cloud hosting
  • Chose Django for its Python base, reusability, automation, and simplicity
  • Implemented Gunicorn and Nginx for a robust foundation

The Learning Process

Throughout this journey, I've embraced various learning methods:

  • Heavily utilized GenAI for coding assistance and decision-making guidance
  • Focused on security best practices
  • Explored cloud technologies, particularly AWS

The Road Ahead

My learning journey is far from over. Next steps include:

  • Learning Ansible and Terraform for system management
  • Making my infrastructure cloud-agnostic
  • Continuing to develop and host web applications

This website is a testament to my journey - from a desire to develop Python web applications to implementing a full-stack solution. It's been an incredible experience, and I'm excited for what's to come!
