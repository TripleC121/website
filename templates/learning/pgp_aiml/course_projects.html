PGP-AIML: My Project Journey

Throughout the PGP-AIML program, I've completed several projects that have allowed me to apply my learning to real-world scenarios. Here's an overview of my project experiences and key takeaways.

Python Foundations

Demonstration of Python programming for data science and Exploratory Data Analysis (EDA) expertise.

Machine Learning

Built my first machine learning model to help a bank marketing department identify potential loan customers.

Advanced Machine Learning

Built multiple advanced machine learning models to classify activities that would keep a customer from giving up their credit card.

Neural Network Optimization

Built multiple Neural Network models to predict customer churn for a bank.

Plant Seedling Classification using Convolutional Neural Networks (CNN)

Developed an advanced image classification system to identify plant seedlings across 12 different species using deep learning techniques, including basic CNNs and transfer learning with VGG16.

Stock Market News Sentiment Analysis and Summarization

Developed an AI-driven sentiment analysis system that will automatically process and analyze news articles to gauge market sentiment, and summarize the news at a weekly level.

Reflections and Future Directions

These projects have been instrumental in solidifying my understanding of AI and ML concepts. They've provided hands-on experience in tackling real-world problems, from data preprocessing to model deployment. Moving forward, I'm excited to delve deeper into advanced topics like reinforcement learning and generative AI, and to apply these skills to more complex, real-world challenges.
